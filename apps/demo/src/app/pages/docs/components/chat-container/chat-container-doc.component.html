ChatContainer

A complete chat interface layout component that combines header, message list, and footer sections.

Installation

import { ChatContainerComponent } from '@angular-ai-kit/core';

Basic Usage

<ai-chat-container
  [messages]="messages()"
  [title]="'AI Assistant'"
  [loading]="isLoading()"
  [showHeader]="true"
  [showFooter]="true"
  [autoScroll]="true"
  [showAvatars]="true"
  (messageCopy)="handleCopy($event)"
  (messageRegenerate)="handleRegenerate($event)"
/>

API Reference

Inputs

Name Type Default Description
messages ChatMessage[] required Array of messages to display
title string '' Title displayed in header
loading boolean false Show loading indicator
showHeader boolean true Show the header section
showFooter boolean true Show the footer section
showAvatars boolean true Show avatars in messages
autoScroll boolean true Auto-scroll on new messages
emptyMessage string 'No messages yet...' Empty state message
customClasses string '' Additional CSS classes

Outputs

Name Type Description
messageSend string Emitted when a message is sent
messageCopy { content: string; message: ChatMessage } Emitted when a message is copied
messageRegenerate ChatMessage Emitted on regenerate click

Features

  • Full-height flex layout
  • Optional header with title and actions
  • Scrollable message area
  • Optional footer for input components
  • Dark mode support
  • Responsive design
