MessageList

A scrollable container for displaying chat messages with auto-scroll, loading states, and empty state handling.

Installation

import { MessageListComponent } from '@angular-ai-kit/core';

Basic Usage

<ai-message-list
  [messages]="messages()"
  [loading]="isLoading()"
  [autoScroll]="true"
  [showAvatars]="true"
  (messageCopy)="handleCopy($event)"
  (messageRegenerate)="handleRegenerate($event)"
/>

Loading State

Shows a skeleton loading indicator when waiting for a response.

Empty State

Displays a customizable message when there are no messages.

API Reference

Inputs

Name Type Default Description
messages ChatMessage[] required Array of messages to display
loading boolean false Show loading/typing indicator
autoScroll boolean true Auto-scroll to bottom on new messages
showAvatars boolean true Show avatars for messages
emptyMessage string 'No messages yet' Message shown when list is empty
customClasses string '' Additional CSS classes

Outputs

Name Type Description
messageCopy { content: string; message: ChatMessage } Emitted when a message is copied
messageRegenerate ChatMessage Emitted when regenerate is clicked on a message

Features

  • Auto-scroll to latest message
  • Skeleton loading indicator while waiting for AI response
  • Empty state with customizable message
  • Smooth scroll behavior
  • SSR-safe scroll handling

Accessibility

  • role="log" for screen reader announcement
  • aria-live="polite" for new message announcements
  • Loading state announced to screen readers
  • Keyboard accessible message actions
