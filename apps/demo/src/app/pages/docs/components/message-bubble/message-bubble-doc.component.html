MessageBubble

A versatile chat message component that displays messages with role-based styling, avatars, and action buttons.

Installation

import { MessageBubbleComponent } from '@angular-ai-kit/core';

Basic Usage

<ai-message-bubble
  [message]="message"
  [showAvatar]="true"
  [showActions]="false"
  (copied)="handleCopy($event)"
  (regenerate)="handleRegenerate()"
/>

With Actions

Action buttons can be always visible or shown on hover.

@if (copiedMessage()) {
Copied to clipboard!
}

Message Roles

Messages are styled differently based on their role: user, assistant, or system.

API Reference

Inputs

Name Type Default Description
message ChatMessage required The chat message to display
showAvatar boolean true Whether to show the avatar
showActions boolean false Always show action buttons (otherwise shown on hover)
customClasses string '' Additional CSS classes to apply

Outputs

Name Type Description
copied string Emitted when copy button is clicked with message content
regenerate void Emitted when regenerate button is clicked (assistant messages only)

ChatMessage Interface

interface ChatMessage {
  id: string;
  role: 'user' | 'assistant' | 'system';
  content: string;
  timestamp: Date;
}

Accessibility

  • Uses semantic article element
  • ARIA labels for screen readers
  • Keyboard accessible action buttons
  • Focus management for hover states
  • Screen reader only text for icon buttons
