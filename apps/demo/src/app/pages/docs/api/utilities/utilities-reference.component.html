Why use cn()?

  • Handles conditional classes cleanly
  • Merges conflicting Tailwind classes correctly
  • Supports arrays, objects, and string arguments
  • Type-safe with excellent DX
 
import {{ '{' }}
  cn,
  formatDate,
  truncateText,
  generateId,
  debounce,
  copyToClipboard,
{{ '}' }} from '@angular-ai-kit/utils';
